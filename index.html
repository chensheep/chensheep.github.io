You Only Look Once: Unified, Real-Time Object Detection

Posted on 2020-07-03 Edited on 2020-07-06 Views: Disqus:

論文請參考arXiv:1506.02640。

介紹

YOLOv1 於 2016 年提出，將 object detection 當成單一回歸問題去解決，分成預測 bounding boxes 與 class possibilities。

與兩階段 R-CNN 系列不同，YOLOv1 使用單一網路，可以 end-to-end 訓練， 因此 YOLOv1 預測速度可達 45 FPS。

更快的同時， YOLOv1 localization errors 雖然更高，但對於背景卻有更低的 false positives 。

YOLOv1 對物體特徵有更好的泛化效果，相比於 DPM 與 R-CNN，當資料集 domain 改變時能有更好的預測效果。

Why do deep convolutional networks generalize so poorly to small image transformations?

Posted on 2020-07-02 Edited on 2020-07-06 Views: Disqus:

論文請參考 arXiv:1805.12177 。

前言

閱讀 STNet 時，論文提到 CNNs 缺乏 spatial invariant ， 但也有一些論文指出 CNNs 對於平移有一定的不變性，因此我進行了簡單的實驗。

使用 MNIST 簡單實驗中，對測試集資料進行微小平移會大幅降低模型預測的準確度，因此找了很多資料，發現這篇論文提供詳細的實驗方法。

論文中一開始提到 CNNs 一般被假定對於微小的圖片轉換有不變性，表示圖片中物體位置位移， CNNs 應能正確分類圖片。

但已有許多證明微小的 translations 或 rescalings 都會劇烈影響網路預測的結果。

這篇論文量化了這個現象，解釋為什麼卷積架構與 data augmenation 沒辦法達到預期的 invariance 效果。

論文中分成幾個部分討論

  • Introduction
  • Quantifying the lack of invariance in modern CNNs
  • Ignoring the Sampling Theorem
  • Why don’t modern CNNs learn to be invariant from data augmentation?

Deformable Convolutional Network (DCNv1)

Posted on 2020-06-25 Edited on 2020-07-06 Views: Disqus:

論文請參考 arXiv:1703.06211。

介紹

論文中提到，由於 CNNs 固定的幾何結構，限制了模型對於幾何轉換的能力，因此提出兩個具體的方法解決這個問題。

具體的方法為 Deformable Convolution 與 Deformable RoI pooling ，藉由增加可學習的 offsets ，調整空間取樣的位置。

兩種做法都能輕易地移植到當今流行的 CNNs 中，使用 Back-Propagation 去訓練 offsets。
